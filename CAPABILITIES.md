## Drawing and Design Tools

### Shapes
Create professional wireframes using various shapes:

- **Rectangle**: Draw rectangular boxes for UI components
- **Circle**: Create circular elements for buttons, avatars, or icons
- **Triangle**: Add triangular shapes for design accents
- **Line**: Draw straight lines to connect elements or create borders
- **Arrow**: Add directional arrows to indicate flow or relationships

### Text Elements
- Add text labels and annotations to your wireframes
- Customize text size and positioning
- Edit text content directly on the canvas

### Brush Tools
Multiple brush options for freehand drawing:

- **Thin Brush**: Fine lines for detailed work (1px width)
- **Medium Brush**: Standard drawing width (5px width)
- **Thick Brush**: Bold lines for emphasis (10px width)
- **Pencil Brush**: Classic pencil-style drawing
- **Circle Brush**: Create dotted or circular patterns
- **Spray Brush**: Airbrush effect for shading and effects

### Element Widgets
Pre-designed UI elements for rapid wireframing:

- **Box**: Standard container elements
- **Text**: Text input fields
- **Image**: Image placeholders
- **Circle**: Circular elements
- **Line**: Various line styles

## User Authentication

### Registration
- **Email-based Registration**: Create an account using email and username
- **Google Sign-In**: Quick registration using Google OAuth
- **Username Requirements**: Usernames must be 4-16 characters, starting with a letter
- **Password Security**: Passwords require at least 8 characters with letters and numbers
- **Email Verification**: Verify your email address after registration

### Login Options
- **Standard Login**: Username and password authentication
- **Remember Me**: Option to stay logged in across sessions
- **Google Sign-In**: One-click login with Google account
- **Session Management**: Secure session handling

### Profile Features
- **Profile Picture**: Upload and customize your profile picture
- **Account Settings**: Manage your account information
- **Password Change**: Update your password securely
- **Logout**: Safely end your session

## Draft Management

### Creating Drafts
- **New Draft**: Create a new wireframe project with a custom title
- **Auto-Save**: Automatic saving every 10 seconds to prevent data loss
- **Manual Save**: Save your work manually at any time

### Draft Operations
- **Load Draft**: Open previously saved drafts
- **Update Draft**: Save changes to existing drafts
- **Delete Draft**: Remove unwanted drafts
- **Draft Dashboard**: View all your saved drafts in one place
- **Edit Existing Draft**: Continue working on saved wireframes

### Draft Features
- Each draft is associated with your user account
- Drafts are stored securely in the database
- Quick access to recent drafts from the dashboard

## Export and Import

### Export Options
- **Export as PNG**: Save your wireframe as a high-quality PNG image
- **Export as JSON**: Export the entire canvas as a JSON file for backup or sharing
- **Download**: Direct download of exported files to your device

### Import Options
- **Import JSON**: Load previously exported JSON files
- **Restore Canvas**: Import restores all elements, styles, and positions
- **File Upload**: Simple file selection interface for importing

## Canvas Controls

### Navigation
- **Cursor Tool**: Default selection and movement tool
- **Pan and Zoom**: Navigate large wireframes easily
- **Element Selection**: Click to select and modify elements
- **Multi-Select**: Select multiple elements at once

### Editing Actions
- **Undo**: Reverse your last action (supports multiple undo steps)
- **Redo**: Reapply undone actions
- **Delete**: Remove selected elements from the canvas
- **Reset**: Clear the entire canvas to start fresh

### Element Manipulation
- **Move**: Drag elements to reposition them
- **Resize**: Adjust element dimensions with control handles
- **Rotate**: Rotate elements to any angle
- **Layer Management**: Elements can be layered on top of each other

### Canvas Features
- Responsive canvas that adapts to window size
- Grid-based layout for precise alignment
- Transparent background option
- Object snapping and alignment helpers

## Getting Started

1. **Register**: Create an account or sign in with Google
2. **Access Editor**: Click "Start Designing" or navigate to the editor
3. **Create Wireframe**: Use drawing tools to build your wireframe
4. **Save Draft**: Give your project a name and save it
5. **Export**: Download your wireframe as PNG or JSON

## Tips for Best Results

- Use the **limited color palette** to focus on structure over aesthetics
- Leverage **auto-save** by keeping your draft open while working
- Take advantage of **undo/redo** to experiment freely
- Use **shapes and widgets** for consistent UI elements
- **Export as JSON** to create backups of important wireframes
- Organize your work with **meaningful draft names**
